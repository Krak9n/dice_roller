# Dice Roller

A small dice roller for terminal.
Just insert the desired value and watch them go.

### Installation
```
git clone https://github.com/Krak9n/dice_roller.git
cd dice_roller
make
make install
```

### Installation with a script
```
git clone https://github.com/Krak9n/dice_roller.git
cd dice_roller
chmod +x install.sh
./install.sh
```

### Usage
Just type in your terminal after Making
```
./throw_dice
```
